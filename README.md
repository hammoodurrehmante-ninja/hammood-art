# Hammood-art

A new creative project for Hammood art compositions.

## Project structure

- `gallery.html` — simple starter gallery page
- `assets/` — place image and media files here
- `sketches/` — save drafts, sketches, and art notes

## Getting started

1. Add your art files to `assets/`.
2. Open `gallery.html` in a browser to preview.
3. Use this folder as the base for your Hammood art work.
